\appendix
\chapter{Reproducibility and Completion Checklist}
\label{chap:appendix}

\section{Demonstration Record}
Each episode should contain:
\begin{itemize}
  \item timestamps for image, tracker, wrench, robot, and command streams;
  \item raw and calibrated tool pose with explicit frame names and transform version;
  \item raw, bias-compensated, filtered, and transformed six-axis wrench;
  \item RGB observations, target-region metadata, and workpiece identifier;
  \item episode boundaries, contact phase, safety events, and acceptance or rejection status;
  \item tool, abrasive, sensor-zeroing, and operator metadata.
\end{itemize}

\section{Implementation Parameters to Report}
\begin{itemize}
  \item VR-to-robot and tracker-to-tool calibration procedures and residual errors;
  \item stream rates, synchronization method, interpolation, filtering, and maximum skew;
  \item image and pose encoders, GRU/LSTM dimensions, history length \(K\), horizon \(H\), fusion and prediction heads;
  \item loss definitions, weights, normalization, optimizer, schedule, split, seed count, and checkpoint rule;
  \item admittance axes, frame, discrete update, \(M_d\), \(D_d\), \(K_d\), contact detection, saturation, and reset behavior;
  \item RL algorithms, observations, actions, rewards, bounds, rates, training environment, randomization, and stopping rule.
\end{itemize}

\section{Experiment Fields to Complete}
\begin{itemize}
  \item robot, polishing tool, force/torque sensor, camera, curved headlight set, and abrasive consumables;
  \item demonstration count, trial count, target-region assignment, randomized order, and termination conditions;
  \item contact threshold, normal estimator, PQI protocol, consistency metric, timing definition, and statistical analysis;
  \item all cells in Tables~\ref{tab:force_action_interface}, \ref{tab:force_trajectory_tracking}, and \ref{tab:residual_rl_ablation};
  \item representative trajectories, force traces, alignment examples, before/after observations, and failure cases.
\end{itemize}

\section{Evidence Status}
The present thesis is an expanded, template-adapted version of the FAIRE review draft. Every numerical result and implementation-specific value absent from that repository remains planned or TBD. Before submission, each placeholder should be replaced by a value traceable to code, configuration, calibration output, experiment log, or inspection record.
